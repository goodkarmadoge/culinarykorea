# Culinary Korea

A static web experience for planning food-led Korea trips around Culinary Class Wars chefs, viral restaurant momentum, and fine-dining destinations.

## Local Preview

```powershell
node server.mjs 4173
```

Then open `http://127.0.0.1:4173`.

## Deploy

This repository is configured for GitHub Pages. Pushes to `main` publish the static site from the repository root.
